This is just a very simple plugin to have a **scroll back to top** button throughout your whole blog/site.

== Description ==
The plugin adds a floating navigation button right bottom in the footer of your site that only appears when you scroll the page down, and when clicked gently roll the site to the top. All this without any modification to your template.

= Full Feature Set =
* Very lightweight, Total content is less than 1 kb and load when the site load
* No dependency, Written in Vanilla JavaScript
* Support all major browsers including Firefox, Chrome, Opera, Edge, IE9 and higher
* Five SVG icon type
* Two icon shape: Square or Circle
* Unlimited Colors for background and icon

== Installation ==
Installing the plugins is just like installing other WordPress plugins. If you don't know how to install plugins, please review the option below:

* From your WordPress dashboard go to **Plugins > Add New**.
* Search for **Scroll to Top** in **Search Plugins** box.
* Find the WordPress Plugin named **Scroll to Top** by **Sayful Islam**.
* Click **Install Now** to install the **Scroll to Top** Plugin.
* The plugin will begin to download and install.
* Now just click **Activate** to activate the plugin.

If you still need help. visit [WordPress codex](https://codex.wordpress.org/Managing_Plugins#Installing_Plugins)
